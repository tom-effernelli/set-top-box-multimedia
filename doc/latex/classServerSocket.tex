\doxysubsection{Detailed Description}
TCP/\+IP IPv4 server socket. Waits for requests to come in over the network. TCP/\+IP sockets do not preserve record boundaries but \mbox{\hyperlink{classSocketBuffer}{Socket\+Buffer}} solves this problem.
